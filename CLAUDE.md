# CLAUDE.MD

## 项目概要
这是一个用于测试 **EPSON DLQ-3500K** 针式打印机的 Java 项目。  
目标是通过 Java 程序向打印机发送 **ESC/P-K** 命令序列，验证打印机的各项功能（文字、汉字、图形、页面控制等）。

## 技术栈与环境
- 语言：Java (建议 JDK17  或以上)
- 与打印机通信方式（待选或按环境配置）：
    - 通过并行端口 (LPT) 直接写入文件流
    - 使用 `javax.print` 标准打印服务 API
    - USB 转串口/并口虚拟端口
    - 网络打印 (Standard TCP/IP Port, 端口 9100)
- 依赖库：
    - 如需串口通信：可选用 JSSC 或 RXTX
    - 目前项目可能为纯 JavaSE，无外部依赖


## 核心任务
- [ ] 建立与 DLQ-3500K 的物理/网络连接
- [ ] 实现底层的原始字节发送功能
- [ ] 封装常用 ESC/P-K 命令的构建方法（走纸、换页、汉字控制等）
- [ ] 编写并运行以下测试：
    - 打印机初始化与状态查询
    - 英文文本打印（不同 CPI、加粗、斜体、下划线）
    - 汉字打印（宋体/黑体，倍宽倍高，纵向打印）
    - 图形（位图）打印
    - 页格式控制（页长、边界、页缝跳过）
- [ ] 验证打印输出的物理效果是否与命令预期一致

## 编程注意事项
- 打印机命令由控制码开头：`ESC` (0x1B), `FS` (0x1C), `GS` (0x1D) 等。
- 命令中的数值参数使用无符号字节，注意 Java 中 `byte` 是有符号的，发送时需转换为 `0~255` 的原始字节。
- 发送数据后可能需要调用 `flush()`。
- 汉字模式需先发送 `FS &` (1C 26) 进入，汉字编码通常为双字节（GB2312/GBK），发送时需注意字节序。
- 文档中常用的命令摘要：

| 命令 | 代码 (Hex) | 说明 |
|------|------------|------|
| 初始化打印机 | 1B 40 | ESC @ |
| 换行 | 0A | LF |
| 换页 | 0C | FF |
| 进入汉字模式 | 1C 26 | FS & |
| 退出汉字模式 | 1C 2E | FS . |
| 设置页长(行) | 1B 43 n | ESC C n |
| 选择10cpi | 1B 50 | ESC P |
| 加粗开/关 | 1B 45 / 1B 46 | ESC E / F |
| 位图打印 | 1B 2A m nL nH data | ESC * |

完整命令集请参考 EPSON 官方《ESC/P 编程手册。
